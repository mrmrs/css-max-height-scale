# css-max-height-scale

Functional CSS for max-height-scale

## Filesize

| File | Size |
|------|------|
| `dist/max-height-scale.css` | 1665 bytes |
| `dist/max-height-scale.min.css` | 1111 bytes (230 Gzipped) |

## Install

```sh
npm install css-max-height-scale
```

## Usage

### Import

```css
@import "css-max-height-scale";
```

### CDN

```html
<link rel="stylesheet" href="https://unpkg.com/css-max-height-scale/dist/max-height-scale.min.css">
```

### Direct

```html
<link rel="stylesheet" href="path/to/css-max-height-scale/dist/max-height-scale.min.css">
```

### Responsive

Responsive variants are available for each class with the following suffixes:

| Suffix | Media Query              |
|--------|--------------------------|
| `-s`   | `min-width: 32em`        |
| `-m`   | `min-width: 48em`        |
| `-l`   | `min-width: 96em`        |

Example: `.mxhi1-m` applies the property at the medium breakpoint and above.

## Building

```sh
npm run build
```

Processes `src/max-height-scale.css` with [Lightning CSS](https://lightningcss.dev) and outputs to `dist/`.

- `dist/max-height-scale.css` — formatted
- `dist/max-height-scale.min.css` — minified
